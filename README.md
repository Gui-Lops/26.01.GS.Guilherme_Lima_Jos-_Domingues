## Monitoramento de Eventos Espaciais - Brasil
## Este projeto é um sistema em Python desenvolvido para simular o processamento de dados provenientes de monitoramento por satélites em diferentes regiões do Brasil. O sistema registra eventos como desmatamento, queimadas e 
## variações climáticas, permitindo a análise de padrões e identificação de regiões críticas.

## Funcionalidades
## Registro de múltiplos eventos ambientais.
## Validação de dados (área > 0 e intensidade entre 1 e 10).
## Armazenamento estruturado em listas.
## Análise estatística dos dados coletados:
## Total de eventos e área afetada.
## Média de intensidade.
## Identificação da região com mais ocorrências.
## Cálculo de densidade média de ocorrências.
## Identificação do evento mais crítico.

## Como Usar
## Certifique-se de ter o Python 3 instalado em sua máquina.
## Execute o script principal:

## python 26.01.GS-Guilherme_Lima-Jose_Domingues-

## Siga as instruções no terminal para inserir os dados dos eventos.
## Ao final, o sistema exibirá um relatório detalhado com as análises realizadas.

## Requisitos da Atividade
## Uso exclusivo de estruturas de controle (if, else, for, while) e listas.
## Utilização de funções nativas como max() e index().
## Proibição do uso de funções definidas pelo usuário ou bibliotecas externas como pandas.
